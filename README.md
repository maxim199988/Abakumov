# Expense Tracker

**Автор:** Абакумов Максим 

## Описание программы

Приложение для учёта личных расходов с графическим интерфейсом. Позволяет добавлять расходы с указанием суммы, категории и даты, фильтровать записи по категории и периоду, подсчитывать общую сумму расходов за выбранный период, сохранять и загружать данные в формате JSON.

## Как использовать

1. Убедитесь, что у вас установлен Python 3.6+.
2. Откройте терминал и перейдите в папку проекта.
3. Запустите приложение:
   python main.py

## Примеры использования

- **Добавление расхода:**
  - Введите сумму (например, 1500).
  - Выберите категорию (например, «Еда»).
  - Введите дату (например, 2024-04-28).
  - Нажмите «Добавить расход».
- **Фильтрация по категории:**
  - Введите «Транспорт» в поле фильтра.
  - Нажмите «Применить фильтр».
